## 项目简介

CS:GO API 项目 基于 Flask 库开发
数据提供范围：饰品价格，个人游戏数据。

## 安装与使用

1. 克隆该仓库到本地：

```shell
git clone https://github.com/NWE-horse/CSGO-API.git
```

进入项目目录：
```shell
cd CSGO-API
```

创建一个虚拟环境
```shell
python -m venv env
```
激活虚拟环境
```shell
env\scripts\activate
```

安装依赖
```shell
pip install -r requirements.txt
```

启动应用：
~~flask run~~因设置port和host无效已被弃用
```shell
python app.py
```
如果看到如下提示，那就证明你已经成功启动了。
